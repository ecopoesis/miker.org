# miker.org

A blog built with Hugo.

## New post

```bash
hugo new posts/my-fancy-new-post.md
```

## Development

1) Install hugo: `brew install hugo`
2) Clone the repo
3) Pull the submodules: `git submodule init && git submodule update`
4) Run the server locally: `hugo server -D`

## Deploy

1) `./publish.sh`
